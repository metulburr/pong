http://pygame.org/project-Pong-2878-4685.html

###arguments
run game with defaults

    python game.py
list help arguments

    python game.py --help
fullscreen

    python game.py -f
difficulty

    python game.py -d hard
screen size

    python game.py -s 1024 800
    


###controls
movement 

    WASD or arrow keys
menu

    mouse or arrow keys
other

    PAUSE = p key
    BACK/MENU = escape key
    SELECT = enter key/left mouse button
